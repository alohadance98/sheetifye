# Troubleshooting

## Common Issues

### 1. Spreadsheet is blank
- **Check the Source**: Ensure the asset path is correct and added to `pubspec.yaml`.
- **Invalid Format**: Sheetify currently supports `.xlsx`. Legacy `.xls` files must be converted first.
- **ProviderScope**: Ensure your app is wrapped in a `ProviderScope`.

### 2. Scrolling is laggy
- **Debug Mode**: Performance is significantly slower in Debug mode. Test in **Profile** or **Release** mode for accurate results.
- **Large Styles**: Extreme amounts of unique cell styling can slow down the initial render.

### 3. Formulas are not showing results
- Sheetify reads the "cached value" of formulas. If the file was saved without calculating results (e.g., generated by some automated scripts), Sheetify might show the cell as empty or showing the raw formula string.
- **Fix**: Open the file in Excel/Google Sheets, save it, and try again.

### 4. Merged cells look weird
- Sheetify supports merged cells, but complex overlapping merges might cause rendering artifacts. Ensure merges do not overlap.

## Getting Help

If you're still stuck:
1. Search the [GitHub Issues](https://github.com/vikaspoute/sheetify/issues).
2. Check the [Example App](../example) for a working implementation.
3. Open a new issue with a reproduction sample.
