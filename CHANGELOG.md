# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-05-12

### Added
- Initial professional release of Sheetify.
- High-performance virtualized spreadsheet rendering engine.
- Native XLSX support (loading from assets, files, memory, and network).
- Bi-directional smooth scrolling for large workbooks.
- Comprehensive selection system (single cell and range).
- Modern, theme-aware UI with dark mode support.
- Fully customizable `SheetifyThemeData` for host app integration.
- Clean architecture foundation for future extensibility.
- Comprehensive developer documentation and gallery example app.
